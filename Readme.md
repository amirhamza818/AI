# AI Games

## Games: 
- AI Tic Tac Toe
- AI Chess
- AI Connect Four

## Technologies Used:
- JavaScript
- HTML
- CSS
- Node.js

## Installation:
1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```bash
   cd AI-Games
   ```
3. Install the dependencies:
   ```bash
   npm install
   ```
4. Start the server:
   ```bash
   npm start
   ```
5. Open your browser and go to `http://localhost:3000` to play the games.
